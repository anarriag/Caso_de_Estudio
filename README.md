# DESARROLLO DEL PORTAFOLIO PARA ESPECIALIDADES:

OBJETIVO:

Mostrar de forma clara y atractiva una experiencia técnica concreta que refleje tus aprendizajes, Habilidades y herramientas desarrolladas durante tu recorrido formativo.

Lo anterior como trabajo final del bootcamp Desarrollo de Aplicaciones Front-End Trainne.

PROYECTO ELEGIDO:

El proyecto que elegí para compartir mi experiencia de aprendizaje concreta fue el BookList del módulo 6 del curso Desarrollo de APlicaciones Front-End Trainne. El cual fue desarrollado mediante el FrameWork Vue.

DESAFIO PRINCIPAL Y JUSTIFICACIÓN DEL PROYECTO ELEGIDO:

En lo personal fue un desafio enorme poder en poco tiempo comprender como trabaja el FrameWork Vue para el desarrollo de interfaces de usuario interactivas en aplicaciones Web modernas. Aunque la unica forma de dominar a cabalidad este FrameWork es trabajar con el de forma constante. lo positivo de todo lo anterior, en espoecial de quienes recien conocemos esta tecnolo´gia es que existe documentacion oficial con la cual uno se puede basar para construir aplicaciones Web del tipo SPA con la ventaja de reiutilizar componentes y lo más importante. Vue se integra muy bien con HTML, Css y JavaScript.

SOLUCIÓN PROPUESTA:

Desarrollar una aplicación web tipo SPA (Single Page Application) llamada BookList, utilizando Vue 3, que permita gestionar libros de forma interactiva y sin recargas de página. Esta solución reemplaza los formularios HTML estáticos por una interfaz dinámica con navegación entre vistas (Inicio, Lista de libros y Detalle), uso de componentes reutilizables y rutas mediante Vue Router.

Además, la aplicación incorpora reactividad para actualizar automáticamente la información al agregar, editar o eliminar libros, y utiliza LocalStorage para mantener los datos en el navegador, junto con Bootstrap 4 para un diseño responsivo y moderno.

HERRAMIENTAS TÉCNICAS UTILIZADA PARA EL DESARROLLO

- FrameWord Vue 3.
- HTML.
- CSS3.
- JavaScript.

PRINCIPALES APRENDIZAJES ALCANZADOS.

Desarrollar la capacidad de crear aplicaciones web modernas tipo SPA utilizando Vue.js, aplicando principios de reactividad, modularidad y navegación dinámica, mediante el uso de componentes reutilizables, manejo de rutas con Vue Router y gestión de datos en tiempo real.

Además, se busca comprender cómo estructurar una aplicación escalable, integrando tecnologías como HTML, CSS, JavaScript y herramientas como LocalStorage, para construir interfaces interactivas, mantenibles y orientadas a una mejor experiencia de usuario.

MÉTRICAS DE IMPACTO LOGRADAS

🔹 Métricas funcionales (cumplimiento de requisitos)

- % de funcionalidades implementadas.
  (Agregar, listar, filtrar, editar, eliminar libros)
- Número de vistas operativas.
  (Inicio, Lista, Detalle → mínimo 3).
- Correcto funcionamiento de rutas dinámicas
  (% de navegación sin errores).

🔹 Métricas de rendimiento.

- Tiempo de carga inicial de la aplicación.
- Tiempo de respuesta en acciones (agregar/eliminar libros).
- Tiempo de renderizado reactivo (actualización sin recarga).
- Número de recargas de página (ideal: 0 en SPA)

🔹 Métricas de aprendizaje (aplicado a nuestro caso)

- Nivel de dominio de Vue.js (básico → intermedio)
- Comprensión de SPA y Vue Router
- Capacidad de implementar reactividad
- Uso de frameworks (Bootstrap, Vue)
